# dsc-prometheus-grafana

## Setup


### dsc-datatool

```bash
git clone https://codeberg.org/DNS-OARC/dsc-datatool.git
cd dsc-datatool
python3 -m venv venv --system-site-packages
. venv/bin/activate
pip install -r requirements.txt
pip install -e . --no-deps
```
